# Handoffs

`/png-handoff` 커맨드로 생성한 팀원 인수인계 문서를 보관하는 폴더입니다.

- 기본 생성 경로: `docs/ai/handoffs/<feature-name>-handoff.md`
- 생성: `/png-handoff` 또는 `/png-handoff <feature-name>`
- 삭제: `/png-handoff-clean <파일명 또는 경로>`

문서에는 Feature Summary · Scope Snapshot · Completed Tasks · Remaining Tasks · Risks & Open Decisions · Next Actions 6개 섹션이 포함됩니다.

> AI 에이전트 핸드오프(다른 AI 도구로 작업 이관)는 이 폴더가 아닌 `docs/ai/04-agent-handoff-prompt-template.md`를 사용하세요.
