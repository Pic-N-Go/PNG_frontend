# docs/guide/api/

**"백엔드와 무엇을 주고받는가"** — API 명세, 요청/응답 형식, 미디어 처리 스펙에 관한 문서.

## 문서 목록

| 파일 | 용도 |
|---|---|
| `auth-api.md` | 인증 API 명세 (로그인·회원가입·토큰) |
| `auth-integration.md` | 인증 API 연동 현황 (화면별 완료/미연동, 백엔드 협의 필요 사항) |
| `photo-upload-spec.md` | 사진 업로드 스펙 (형식·크기·EXIF 처리) |

## 포함 기준

- 엔드포인트 URL, 요청/응답 구조 등 **API 계약** 문서
- 미디어 처리 형식·크기·메타데이터 스펙, 외부 서비스 연동 명세
