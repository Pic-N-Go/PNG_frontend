# FAQ 더보기 화면 작업 요청

## 진입 경로
- **마이페이지** → **FAQ 전체 보기** → 특정 항목의 **더보기** 탭
- 스택: `MyPage` → `Settings` → **`FAQ`** (신규)
- 뒤로가기 시 이전 화면(설정)으로 복귀

## 주요 스펙

### 레이아웃 (모바일 390×844 기준)
- 상단 네비 헤더: `height 52` · 좌측 `IconChevronLeft` (40×40 히트영역) · 중앙 타이틀 **"자주 묻는 질문"** (17px / weight 600)
- 콘텐츠 좌우 패딩: **28px**
- 검색바: 높이 44, `#f5f5f7`, 라디우스 12, 좌측 `IconSearch`, placeholder "궁금한 내용을 검색해 보세요"
- 카테고리 칩(1b): 가로 스크롤, 높이 34, pill(999), 비활성 `#fff + 1px rgba(0,0,0,0.08)` · 활성 `#e31b59 / #fff`
- FAQ 카드: 배경 `#f5f5f7`, 라디우스 16, 패딩 16/18, 항목 간 gap 10
- Q/A 뱃지: 22×22 라디우스 6 · Q 활성 `#e31b59/#fff` · Q 비활성 `rgba(0,0,0,0.06)/rgba(0,0,0,0.48)` · A `#fff + 1px 보더`
- 하단 문의 CTA: 카드 배경(`#f5f5f7`)에 우측 pill 버튼(`#e31b59`, 높이 36) — 스크롤 위에 고정

### 컬러 토큰 (마이페이지 계승)
| 용도 | 값 |
|---|---|
| 페이지 배경 | `#ffffff` |
| 카드/인풋 배경 | `#f5f5f7` |
| 브랜드/포인트 | `#e31b59` |
| 텍스트 1차 | `#000000` |
| 텍스트 2차 | `rgba(0,0,0,0.48)` |
| 텍스트 3차/비활성 | `rgba(0,0,0,0.28)` |
| 구분선 | `rgba(0,0,0,0.06)` |

### 카테고리
`전체 / 촬영 스팟 / 사진·업로드 / 계정·보안 / 이용안내 / 기타`
> 사진 촬영자를 위한 장소 정보 앱이므로 결제·환불 카테고리는 사용하지 않음. 최종 항목은 CMS 연동 시 서버 값 사용.

### 타이포
- 헤더 타이틀 17 / 600
- 섹션 라벨 15 / 600
- 질문 14 (열림 600, 접힘 500)
- 답변 13 / rgba(0,0,0,0.68) / lineHeight 1.55
- 카테고리 카운트, 서브 텍스트 12 / text2

## 상태 규칙
- **아코디언**: 탭 시 인라인 확장. **다중 오픈 허용** (Set으로 관리).
- **Q 뱃지**: 오픈 상태에서만 `#e31b59` 채움, 아이콘도 `IconChevronUp`으로 스왑.
- **검색**: `question` + `answer` 모두 소문자 포함 매칭. 실시간 필터.
- **카테고리 선택**: 단일 선택. 기본값 `전체`. 카테고리 + 검색어는 AND 결합.
- **빈 상태**: 결과 0개 → "검색 결과가 없어요 / 다른 키워드로 다시 검색해 보세요."
- **문의하기 CTA**: `navigation.navigate('Inquiry')` — 1:1 문의 스택 진입.
- **접근성**: 각 카드에 `accessibilityRole="button"` + expanded 상태 힌트 (RN 구현 시 추가 권장).

## 유의사항
- 스타일은 인라인 `style={{}}` + `normalize()` 관례를 유지 (StyleSheet.create 사용 금지).
- 아이콘은 `@tabler/icons-react-native` 사용, strokeWidth 2 고정.
- 카드 그림자/보더 사용 금지 — 마이페이지는 flat + 배경색 계층으로 위계를 만듦.
- 애니메이션은 `LayoutAnimation.easeInEaseOut` (Android는 experimental flag 이미 초기화).
- 폰트 weight는 600 이하로만 사용 (기존 규칙).
- 페이지 좌우 패딩 28은 스크롤 뷰 `contentContainerStyle`에 부여 — 카드 그리드 20 규칙은 이 화면에는 적용되지 않음(리스트 화면).

## 시안 (FAQ.dc.html)
- **1a — 아코디언 단일 리스트**: 카테고리 없이 인기순. 항목이 10개 내외라면 충분.
- **1b — 카테고리 탭 + 아코디언** *(권장)*: 촬영 스팟 / 사진·업로드 / 계정·보안 / 이용안내 / 기타로 그룹핑. 마이페이지 설정 화면의 섹션 그룹핑 패턴과 일치.
